LIST COLLECTIONS


get https://api.pinecone.io/collections

This operation returns a list of your Pinecone collections.

Serverless and starter indexes do not support collections.


Responses
200

THIS OPERATION RETURNS A LIST OF ALL THE COLLECTIONS IN YOUR CURRENT PROJECT.



Response body
object

The list of collections that exist in the project.



collections
array of objects
collections
object

The CollectionModel describes the configuration and status of a Pinecone
collection.



name
string
required

size
int64

The size of the collection in bytes.


status
string
required

The status of the collection.

Initializing Ready Terminating


dimension
int32

The dimension of the vectors stored in each record held in the collection.


vector_count
int32

The number of records stored in the collection.


environment
string
required

The environment where the collection is hosted.
