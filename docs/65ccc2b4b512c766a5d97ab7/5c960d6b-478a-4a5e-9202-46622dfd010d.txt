UNDERSTANDING SUBSCRIPTION STATUS



OVERVIEW

This document describes the different subscription statuses for your Pinecone
organization.


SUBSCRIPTION STATUSES FOR ACCOUNTS WITH PAST DUE PAYMENTS

Users on the pay regular payments . When a payment is past due, Pinecone may
restrict your account.

Past due accounts have one of the following subscription statuses:

 * Restricted. This organization is now on the Starter Plan and can no longer
   create new resources.
 * Suspended. The API keys for this organization are no longer valid; calls to
   the Pinecone API fail.
 * Deactivated. Pinecone archives this organization's indexes.


REINSTATING A DEACTIVATED ORGANIZATION

If your organization is deactivated, follow these steps to reinstate your
subscription and reactivate your indexes:

 1. Pay all past due invoices.
 2. (For organizations with cloud marketplace billing) Reenable cloud

 3. In the Pinecone console, back to the desired plan.
 4. Click API Keys. This view displays your new API keys.
 5. Update all queries to use these new keys.
