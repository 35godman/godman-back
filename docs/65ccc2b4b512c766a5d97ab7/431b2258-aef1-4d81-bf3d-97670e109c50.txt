DESCRIBE AN INDEX


get https://api.pinecone.io/indexes/{index_name}

Get a description of an index.

Path Params
index_name
string
required

The name of the index to be described.


Responses
200

CONFIGURATION INFORMATION AND DEPLOYMENT STATUS OF THE INDEX.



Response body
object

The IndexModel describes the configuration and status of a Pinecone index.



name
string
required

The name of the index. Resource name must be 1-45 characters long, start and end
with an alphanumeric character, and consist only of lower case alphanumeric
characters or '-'.


dimension
int32
required

The dimensions of the vectors to be inserted in the index.


metric
string
required

The distance metric to be used for similarity search. You can use 'euclidean',
'cosine', or 'dotproduct'.

cosine euclidean dotproduct


host
string

The URL address where the index is hosted.


spec
object
required
pod
object

Configuration needed to deploy a pod-based index.

pod object
serverless
object

Configuration needed to deploy a serverless index.

serverless object
status
object
required
ready
boolean
required

state
string
required

Initializing InitializationFailed ScalingUp ScalingDown ScalingUpPodSize
ScalingDownPodSize Upgrading Terminating Ready
