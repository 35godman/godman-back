КЛАССЫ SAE

В принципе, все моторные масла выполняют одну и ту же функцию. Они смазывают подвижные части в двигателе, предотвращают износ и
коррозию, а также обеспечивают отвод тепла. Однако каждый двигатель составлен по-разному и требует особых свойств масла, чтобы
гарантировать оптимальную смазку. Несколько институтов выпустили различные классификации и спецификации, чтобы обеспечить
пользователям обзорность.

Вязкость как мера внутреннего трения жидкости является одним из самых важных свойств масла. Она определяет, насколько тонкое или
вязкое масло и как оно себя ведет в зависимости от окружающей температуры. Поэтому SAE (Общество автомобильных инженеров)
опубликовало систему, которая определяет опорные температуры, пределы вязкости и классификации для моторных масел. Существуют
моноградные масла, которые соответствуют определенной кинематической вязкости при 100 °C, такие как SAE 30, которые использовались
многие годы назад, когда масло в автомобилях меняли зимой и летом. Многие классические автомобили до сих пор зависят от
моноградных масел. Современные легковые автомобили работают с мультиградными маслами, такими как SAE 5W30. Мультиградные масла
имеют хорошие свойства текучести зимой и летом и могут использоваться круглый год. Число перед "W" определяет поведение масла при
низких температурах. Чем меньше число, тем более жидким остается масло при низких температурах. Это позволяет моторному маслу
быстро смазывать все важные части в двигателе даже зимой при холодном запуске. Число после "W" определяет кинематическую вязкость
масла при 100 °C. Чем выше число, тем более устойчиво масло к экстремальным нагрузкам.

ТЕХНИЧЕСКИЕ ХАРАКТЕРИСТИКИ СОГЛАСНО ACEA

ACEA расшифровывается как Association des Constructeurs Europeens de l'Automobile (Ассоциация европейских производителей
автомобилей). Она разрабатывает спецификации для моторных масел на основе самых строгих европейских испытательных процедур. Таким
образом, спецификации ACEA представляют собой наибольший общий знаменатель требований европейских производителей автомобилей к
моторным маслам. Эти спецификации подразделяются на последовательности A, B и C. Последовательность E существует исключительно для
грузовиков. Они дополняются цифрами и дополнительно дифференцируются.

Последовательность A - описывает требования для бензиновых двигателей легковых автомобилей. Последовательность B - описывает
требования для дизельных двигателей легковых автомобилей, микроавтобусов и пикапов. Последовательность C - транспортные средства,
оснащенные передовыми системами очистки выхлопных газов (например, дизельными частицами фильтрами), требуют специальных моторных
масел для предотвращения засорения дизельных частицных фильтров. В зависимости от допустимой доли образующих пепел веществ -
сульфатной золы, фосфора и серы, подходящие масла разделяются на Low и Mid SAPS масла и классифицируются в соответствии с ACEA C
спецификацией. Последовательность E - описывает требования для дизельных двигателей грузовиков.

Моторные масла для легковых автомобилей всегда указываются как комбинация бензиновых и дизельных двигателей (например, ACEA
A3/B4). Число после буквы описывает разницу в характеристиках моторных масел. В таблице ниже мы собрали для вас все еще
действующие спецификации ACEA.

ВЯЗКОСТЬ HTHS

Современные двигатели требуют масел с пониженной вязкостью при высоких температурах. Это свойство характеризуется термином HTHS
(High-Temperature-High-Shear-Viscosity - высокотемпературная высокоскоростная вязкость). С использованием таких масел вязкость при
высоких температурах снижается, что уменьшает потери трения в двигателе. Топливо используется более эффективно, что приводит к
снижению расхода топлива для достижения той же мощности двигателя.


СПЕЦИФИКАЦИИ СОГЛАСНО API

API расшифровывается как American Petroleum Institute. Классы API описывают американские требования и критерии качества для
моторных масел. Спецификация для моторных масел разделена на API S, API C и API F.

Буква "S" (Service) означает бензиновые двигатели легковых автомобилей. Буква "C" (Commercial) определяет требования для дизельных
двигателей коммерческих автомобилей и грузовиков. В конце 2016 года для коммерческих автомобилей был дополнительно введен стандарт
API F. Текущая спецификация API CK-4 была разработана с целью обеспечить обратную совместимость и для масел, HTHS значение которых
> 3,5 мПа * с. Масла с релизом API FA-4 используются в более современных двигателях. HTHS значение находится между 2,9 и 3,2 мПа *
с. Основное внимание уделяется дальнейшему снижению выбросов.

Дизельные двигатели легковых автомобилей не классифицируются по API, так как они практически не используются в США. Буквы S и C
постоянно дополняются дополнительными буквами или цифрами для обновления стандартов. В настоящее время API SP и API CK-4 являются
самыми современными спецификациями. Они обеспечивают обратную совместимость со старыми стандартами.
