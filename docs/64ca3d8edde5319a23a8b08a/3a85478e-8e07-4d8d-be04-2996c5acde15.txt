Транспортное средство: Yamaha 30HWC

Кукую присадку использовать для лодочного2-х тактного двигателя. Температура в камере сгорания лодочного мотора ниже сухопутных аналогов из-за чего при использовании обычного 2-х тактного масла образуется нагар (не успевает прогореть). Есть ли присадка позволяющая уменьшить нагар в камерах сгорания.

Здравствуйте, Ярослав! При использовании обычного двухтактного масла будет образовываться нагар. Именно по этой причине, для лодочных моторов разработано своё масло, которое по свойствам ощутимо отличается от обычного двухтактного масла. Для очистки нагара можно использовать очиститель топливной системы бензин. Для системы впрыска рекомендуем периодическое применение присадки СГА.
