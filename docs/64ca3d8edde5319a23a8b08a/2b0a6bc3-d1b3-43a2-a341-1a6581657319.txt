Синтетическое моторное масло «Suprotec Atomium» 5W-30 4л, 1 л.

Полностью синтетическое моторное масло для всех типов современных бензиновых и дизельных двигателей. Классификация SAE 5W-30, ACEA C3, API SN.

Suprotec Atomium создано специально для российских условий. Разрабатывали рецептуру немецкие специалисты по детальному техническому заданию «СУПРОТЕК».

85% состава масла составляет синтетическая ПАО (поли-алфа-олефиновая) основа.
5% - эстеровые масла.
10% - пакет химических присадок.
Не содержит продуктов гидрокрекинга.

За счет отсутствия гидрокрекинговых масел Suprotec Atomium имеет очень высокие эксплуатационные характеристики:

вязкостные - не теряет вязкости в широком диапазоне температур, не теряет текучести до - 40°С;
трибологические - надежно снижает износ двигателя при работе с некачественным топливом, в условиях повышенных нагрузок, движении в пробках;
ресурсные - мало испаряется, обладает высокой стойкостью к окислению, не теряет рабочих совйств на пробегах до 15-20 тысяч километров.

Подробнее о свойствах масла рассказно во вкладке «Применение». Об особенностях перехода на масло Suprotec Atomium с других сортов масел читайте во вкладке «Инструкция»

Для двигателей с большим пробегом, а также для двигателей, которым рекомендованы масла повышенной вязкости «Супротек» предлагает масло Suprotec Atomium 5W-40

Масло SUPROTEC Atomium не содержит триботехнических добавок, являющихся фирменным продуктом «Супротек». Масло обеспечивает снижение трения и скорости износа двигателя, но не обладает восстанавливающими свойствами. Для возвращения рабочих характеристик уже изношенного двигателя необходимо использовать триботехнические составы серии «Актив».

Масло SUPROTEC ATOMIUM может быть использовано и для двигателей предыдущих поколений, в том числе на автомобилях с большим пробегом. Рекомендации по замене масла предыдущей марки на SUPROTEC ATOMIUM смотрите на вкладке «Примечания»

Масло SUPROTEC ATOMIUM произведено в Германии по особой рецептуре, которая разработана немецким концерном ROWE MINERALOELWERK по техническому заданию НПТК «СУПРОТЕК». 

Рецептура включает в себя инновационный пакет присадок и высококачественную основу, которая состоит из полностью синтезированных поли-альфа-олефинов, и не менее 5% эстеров - сложных эфиров. Не содержит продуктов гидрокрекинга.

Созданное по этой рецептуре масло полностью удовлетворяет повышенным требованиям техзадания к триботехническим, температурным и ресурсным свойствам, а также отвечает особенностям российского рынка: климатическим условиям, качеству топлива, сложным режимам езды по загруженным трассам.

Испытания моторного масла «SUPROTEC» в РГУ нефти и газа (НИУ) имени И.М. Губкина.
