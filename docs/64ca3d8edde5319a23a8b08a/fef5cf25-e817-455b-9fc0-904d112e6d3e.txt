Трансмиссионное масло SUPROTEC ATOMIUM класса API GL – 4/5 разработано для:

механических коробок передач;
раздаточных коробок и дифференциалов современных легковых автомобилей.

Внимание! Трансмиссионное масло SUPROTEC ATOMIUM не подходит для использования в ступенчатых передачах DCT/DSG и в бесступенчатых коробках передач. 

Соответствует допускам:

API GL-4/GL-5/MT-1
MIL-L-2105D/MIL-PRF-2105E
SAE J2360
MAN 341 Type E3/3343 Type S
MB 235.8
SCANIA STO 1:0
ZF TE-ML 02B, 05B, 07A, 08, 12B, 16F, 17B, 19C, 21B 
 

Кроме того, рекомендуется к применению при спецификациях:

Ford SQM-2C-9002 AA/M2C200-C
GM 1940182/1940768/M75/1
MAN 341 Type E3/3343 Type S
MB 235.0/235.72
VW G 50/51 (VW 501.50/51)/G052 190/G052 911
ZF TE-ML 05A, 12E, 16B, 19B, 21A

Трансмиссионное масло SUPROTEC ATOMIUM может быть использовано и для трансмиссий предыдущих поколений, в том числе на автомобилях с большим пробегом. Рекомендации по замене масла предыдущей марки на SUPROTEC ATOMIUM смотрите на вкладке «Примечания»

Трансмиссионное масло SUPROTEC ATOMIUM произведено в Германии по особой рецептуре, которая разработана немецким концерном ROWE MINERALOELWERK по техническому заданию НПТК «СУПРОТЕК». 

Рецептура включает в себя инновационный пакет присадок и высококачественную основу, которая состоит из полностью синтезированных поли-альфа-олефинов, и не менее 5% эстеров - сложных эфиров. Не содержит продуктов гидрокрекинга.

Созданное по этой рецептуре трансмиссионное масло полностью удовлетворяет повышенным требованиям техзадания к триботехническим, температурным и ресурсным свойствам, а также отвечает особенностям российского рынка: климатическим условиям, качеству топлива, сложным режимам езды по загруженным трассам.

Подробнее о преимуществах трансмиссионного масла SUPROTEC ATOMIUM смотрите на вкладке «Описание».
